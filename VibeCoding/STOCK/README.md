# EQV 핵심 인사이트 (주식 분석 웹/앱) 설치 및 실행 가이드

현재 제작하신 이 프로젝트는 **웹 브라우저**에서 동작할 뿐만 아니라, **바탕화면의 독립된 앱**으로 설치하거나 **데스크톱 프로그램(Electron)**으로도 실행할 수 있도록 하이브리드 형태로 구성되어 있습니다.

아래의 3가지 방법을 통해 원하시는 형태로 실행하실 수 있습니다.

---

## 1. 브라우저에서 개발 서버로 실행하기 (가장 기본)
코드를 수정하거나 새로운 기능을 테스트할 때 가장 많이 사용하는 방법입니다.

1. **터미널(명령 프롬프트) 열기**
   - 개발 환경(VS Code 등)에서 터미널을 열고, 프로젝트 폴더(`C:\Workspace\VibeCoding\STOCK`)로 이동합니다.
2. **필수 패키지 설치** (최초 1회만)
   ```bash
   npm install
   ```
3. **서버 실행**
   ```bash
   npm run dev
   ```
4. **접속하기**
   - 터미널에 나타나는 주소 (예: `http://localhost:5173/`)를 `Ctrl`을 누른 채 클릭하거나 인터넷 창에 복사하여 붙여넣기 하면 앱이 열립니다.

---

## 2. 인터넷 브라우저에서 '앱(App)'으로 설치하기 (PWA 기능)
방금 만들어드린 **아이콘**을 활용하여, 크롬(Chrome)이나 엣지(Edge) 브라우저에서 마치 일반 프로그램처럼 설치해 바탕화면에 바로가기를 만들 수 있습니다.

1. 위 1번 방법(`npm run dev`)으로 브라우저에서 앱을 엽니다.
2. 브라우저 상단의 **주소창 우측 끝**을 확인합니다.
   - **Chrome**: 주소창 끝에 **[💻 앱 다운로드]** 또는 **[설치]** 아이콘이 나타납니다.
   - **Edge**: 주소창 끝에 **[앱 사용 가능]** 아이콘이 나타납니다.
3. 해당 아이콘을 클릭하고 **설치**를 누릅니다.
4. **결과**: 브라우저의 탭이 사라지고, 독립된 창(Window) 형태로 앱이 실행됩니다. 동시에 PC 바탕화면과 시작 메뉴에 방금 만든 멋진 아이콘으로 바로가기가 생성됩니다!

---

## 3. 데스크톱용 설치 파일(exe)로 완전히 빌드하기 (Electron)
이 프로젝트에는 데스크톱 백엔드(Electron) 코드가 포함되어 있습니다. 이를 통해 완전히 독립된 `.exe` 윈도우 프로그램으로 만들 수 있습니다.

1. **개발 모드로 윈도우 창 띄우기**
   ```bash
   npm run electron:dev
   ```
   *(이 명령을 치면 브라우저가 아닌 별도의 프로그램 창이 바로 뜹니다.)*

2. **최종 배포용 exe 파일 만들기 (빌드)**
   ```bash
   npm run electron:build
   ```
   - 이 명령어가 완료되면 프로젝트 폴더 내부에 `dist` 또는 `out` 폴더가 생기며, 그 안에 다른 사람에게 공유하거나 설치할 수 있는 `.exe` 실행 파일이 생성됩니다.
